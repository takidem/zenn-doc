# git gc

https://alpacahack.com/daily/challenges/git-gc

## 今回使ったコマンド

```bash
$ git fsck

$ git cat-file -p <hash>
```

## 解いてみた

問題用のgzファイルをダウンロードします。

```bash
$ ll
total 12K
-rwxrwxrwx 1 takidem takidem 11K Jan 26 21:23 git-gc.tar.gz

# ファイル形式を見てみます
$ file git-gc.tar.gz
git-gc.tar.gz: gzip compressed data, from Unix, original size modulo 2^32 61440

# ただのgzのようなので解凍して中身を確認します
$ tar -xzf git-gc.tar.gz
$ ll
total 16K
drwxr-xr-x 3 takidem takidem 4.0K Jan 10 01:45 git-gc
-rwxrwxrwx 1 takidem takidem  11K Jan 26 21:23 git-gc.tar.gz

# 移動してgitの状態を確認してみます
$ ll
total 8.0K
drwxr-xr-x 3 takidem takidem 4.0K Jan 10 01:45 chall
-rwxr-xr-x 1 takidem takidem  313 Jan 10 01:13 generate.sh
```
なんだか見てほしそうなシェルがあるので、開いてみます。
![](https://storage.googleapis.com/zenn-user-upload/fccd6f4254cb-20260126.png)

flag.txt をaddしてますね。

これはgitの履歴を追っていけば、残存しているgitのオブジェクトから元ファイルをとってこれるかもです
```bash
# ログを確認
git log
commit c0bf20c191a250522fc742093ff920243346f578 (HEAD -> main)
Author: AlpacaHack <alpacahack@alpacahack.internal>
Date:   Sat Jan 10 01:45:04 2026 +0900

    initial commit

# commitされていそうですね
```
過去のcommitを見るには、ダングリングコミットを確認するのがよさそうな気がするので、fsckを打ってみます
```bash
$ git fsck --no-reflogs
Checking object directories: 100% (256/256), done.
Checking objects: 100% (5/5), done.
dangling commit 75a6ad9f0abe942df11f90b58f175d553c23c101
Verifying commits in commit graph: 100% (1/1), done.
```
commitした形跡が残っていました。
では、これのハッシュ化された文字列を解析してみます。
```bash
$ git cat-file -p 75a6ad9f0abe942df11f90b58f175d553c23c101
tree 1b44243c75077537d74f00aacbc930f2c7283b93
parent c0bf20c191a250522fc742093ff920243346f578
author AlpacaHack <alpacahack@alpacahack.internal> 1767977104 +0900
committer AlpacaHack <alpacahack@alpacahack.internal> 1767977104 +0900

add flag
```
add flagというコメントが残っていて、generate.shの中身で追加されていそうなことが分かります。
では、このtreeになっているところ再度解析してみます。
```bash
$ git cat-file -p 1b44243c75077537d74f00aacbc930f2c7283b93
100644 blob 4c820f244ddd11f3286edfb63ac9d1537a3f2cb3    flag.txt
```
blobの中身が残存していました。
さらに解析・・・
```bash
$ git cat-file -p 4c820f244ddd11f3286edfb63ac9d1537a3f2cb3
Alpaca{--prune=now_1s_4ll_y0u_n33d}
```

はい、フラグが出てきました。
gitコマンド知っていれば、なんとなくで解ける感じになっている問題でした。
